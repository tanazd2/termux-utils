# Security Policy

## Supported Versions
We actively maintain the following versions of this repository:

| Version | Supported          |
|---------|------------------|
| Main    | ✅ Yes            |
| Previous releases | ⚠️ Limited support |

## Reporting a Vulnerability
If you discover a security vulnerability in this repository, please open a new issue.

## Security Updates
Once a vulnerability is confirmed:
- A fix will be prepared and released as soon as possible.
- The release notes will mention the security fix.
- Users are encouraged to update to the latest version immediately.

## Reporting Guidelines
- Please include **all relevant information** to help us reproduce and fix the issue quickly.
- Do **not** attempt to exploit the vulnerability on users’ systems. Only test in controlled environments.

Thank you for helping us keep this project secure!
